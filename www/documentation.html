Documentation for the Interactive Graph Plotting using ggplot2" ShinyApp

An Overview

The purpose of this Shiny project to provide an interactive processing and plotting of a graph with the ggplot2 package available in R and Shiny Application. This application uses the 38 model of motocars data with reference to the mpg fuel economy data from 1999 and 2008. There are 5 selection for the graph plotting: the type of graph, x value, the y value, the colours and the facet_wrap value. Changes made to any of the 5 selections will change the graph display.

How to Use

  1. Change the selection in any of the 5 selection input
  2. View how the displayed graph changes
  3. View the updated ggplot command displayed below the plot
